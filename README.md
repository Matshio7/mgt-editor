### Installation
```bash
# Clone the repository
git clone https://github.com/mcfly/mgt-editor.git
cd mgt-editor

# Install dependencies
npm install

# Build game data assets
npm run build:assets

# Start development server
npm run hot
```

### Build for Production
```bash
npm run build
```
